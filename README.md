# ✅ Simple To-Do List App

A minimal To-Do list Android app built using Jetpack Compose.  
Allows users to:
- Add tasks
- View tasks in a list
- Delete tasks by tapping

## 🔧 Built With
- Kotlin
- Jetpack Compose
- Android Studio
